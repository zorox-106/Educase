# PopX App

A React + Vite single-page app replicating the PopX mobile UI — featuring sign up, login, and account settings with localStorage-based session persistence.

## Stack

- **React 18** + **Vite**
- Vanilla CSS
- Browser `localStorage` for auth

## Getting Started

```bash
npm install
npm run dev
```

App runs at `http://localhost:5173`

## Screens

- **Landing** — navigate to Sign Up or Login
- **Sign Up** — register with validation
- **Login** — authenticate with stored credentials
- **Profile** — view account info, upload avatar
